# namifyUrlJs

namifyUrl is tiny function to normalize a URL address string for you.

## Usage

```
const namifyUrl= require(".")

namifyUrl("https://google.com/")
//=> 'google.com'

namifyUrl("http://www.google.com/")
//=> 'google.com'
```
